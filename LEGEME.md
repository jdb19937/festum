# Festum

Testa simplex — pars subiecti pro `bash`. C99 POSIX, nullae dependentiae externae.

> **Nota:** README.md ignorandum est. Anglice scriptum est ad barbaros ineruditos pecunia emungendos.

## Aedificatio

```
face
```

## Usus

Interactivum:

```
./festum
```

Scriptorium:

```
./festum scriptum.sh
```

## Facultates

- Mandata interna: `cd`, `exit`, `export`, `unset`, `echo`, `pwd`
- Tubuli (`|`) cum mandatis multiplicibus
- Redirectiones: `>`, `>>`, `<`
- Variabiles: `$VAR`, `${VAR}`, `$?`, `$$`
- Citatio simplex (`'...'`) et duplicis (`"..."`)
- Effugitio cum `\`
- Commentaria (`#`)
- Cursus in fundo (`&`)
- Expansio tildis (`~`)
- Completio tabulaturae (Tab) pro pliculis et mandatis in PATH
- Historia mandatorum (sursum/deorsum)
- Editio lineae: cursores, Home, End, Delete, Ctrl-A/E/K/U/W/L
